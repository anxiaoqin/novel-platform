# Novel Craft Kit · 安装与使用说明

## 方式一：作为扣子技能安装（推荐）

1. 打开扣子Bot编辑页面 → 添加技能 → 从文件安装
2. 上传 `novel-craft-kit` 整个文件夹
3. 安装后Bot即可使用评分审核、创作指导、去AI化改写能力

## 方式二：作为知识库导入

1. 扣子知识库页面 → 创建知识库 → 上传 `references/` 下所有md文件
2. 关联到Bot，在系统提示词中说明引用规则

## 方式三：直接复制文件

将 `references/` 下文件复制到工作目录，对话中让AI读取即可
