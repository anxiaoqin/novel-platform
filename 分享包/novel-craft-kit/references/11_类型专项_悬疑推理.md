# 类型专项·悬疑推理

> 本指南针对悬疑、推理小说的特有创作要点。

---

## 一、悬疑推理核心原则

1. **逻辑是骨架**：推理必须自洽，漏洞是致命伤
2. **悬念是引擎**：没有悬念就没有阅读动力
3. **公平是底线**：线索必须提前给出
4. **意外是高潮**：转折必须出乎意料
5. **人性是深度**：悬疑背后是人性

---

## 二、悬疑推理创作要点

### 2.1 悬疑小说类型

| 类型 | 特点 | 核心要素 |
|------|------|---------|
| **本格派** | 解谜为核心 | 公平线索、逻辑推理 |
| **社会派** | 犯罪动因为核心 | 社会问题、犯罪动机 |
| **硬汉派** | 侦探个人魅力 | 行动、氛围、人物 |
| **惊悚悬疑** | 恐怖元素 | 心理压迫、恐惧感 |
| **心理悬疑** | 心理游戏 | 人性幽暗、复杂动机 |
| **推理惊悚** | 推理+惊悚 | 推理逻辑+恐怖氛围 |

### 2.2 核心谜题设计

**【谜题类型】**：

```
谜题 = 谁 + 做了什么 + 如何做到 + 为什么

谁（Whodunit）：
- 凶手是谁
- 嫌疑人是谁

做了什么（What）：
- 发生了什么案件
- 案件的性质

如何做到（How）：
- 犯罪手法
- 作案过程
- 破解关键

为什么（Why）：
- 犯罪动机
- 深层原因
```

**【好谜题的特征】**：
- 有解但不容易
- 线索公平给出
- 有社会/人性深度
- 解答意外但合理

### 2.3 悬念设计

**【悬念类型】**：

| 悬念类型 | 描述 | 效果 |
|---------|------|------|
| **谜题悬念** | 核心问题未解答 | 读者想知道答案 |
| **命运悬念** | 角色命运不确定 | 读者担心角色 |
| **秘密悬念** | 角色有隐藏秘密 | 读者想揭露 |
| **冲突悬念** | 即将发生的冲突 | 读者期待 |
| **认知悬念** | 读者不知道角色知道什么 | 读者想了解更多 |

**【悬念节奏设计】**：

```
悬念节奏 = 建立 → 维持 → 深化 → 解答

建立（10-20%）：
- 抛出问题
- 吸引注意
- 创造期待

维持（40-60%）：
- 维持好奇
- 给出部分信息
- 增加新的问题

深化（10-20%）：
- 问题升级
- 压力增大
- 接近临界点

解答（10-20%）：
- 揭示答案
- 情感释放
- 意义升华
```

## 三、推理逻辑设计

### 3.1 推理基础结构

**【推理结构】**：

```
推理 = 线索 → 推理 → 结论

线索：
- 必须是公平的
- 必须是可见的
- 必须是有意义的

推理：
- 必须是逻辑的
- 必须是合理的
- 必须是可验证的

结论：
- 必须是唯一的
- 必须是意外的
- 必须是合理的
```

### 3.2 线索设计

**【线索设计原则】**：

```
线索设计 = 可见 + 有意义 + 可组合

可见：
- 读者必须能看到
- 不能只存在于凶手视角
- 提前给出，不要临时添加

有意义：
- 与谜题相关
- 能推动推理
- 组合后指向真相

可组合：
- 多条线索可以组合
- 线索有不同层次
- 最终组合指向真相
```

**【线索分配】**：

```
线索分配公式：
凶手知道（100%）= 读者知道（50%）+ 角色知道（30%）+ 隐藏（20%）

读者必须知道的线索：
- 推理的核心线索
- 至少提前1/3篇幅给出

角色知道的线索：
- 侦探/主角视角
- 推动调查

隐藏的线索：
- 次要线索
- 误导线索
- 最终揭秘时揭示
```

**【AI指令模板】**：

```
请设计推理小说的线索系统：

【谜题】
- 核心问题：[描述]
- 凶手：[描述]
- 手法：[描述]
- 动机：[描述]

【线索清单】
| 线索 | 内容 | 类型 | 给出时机 | 给谁 |
|------|------|------|---------|------|
| 线索1 | [描述] | 核心/辅助/误导 | 第X章 | 读者/角色 |
| 线索2 | [描述] | ... | ... | ... |

【公平性检查】
- [ ] 核心线索是否提前给出？
- [ ] 线索是否只给读者一次？
- [ ] 读者是否能自己推理？
```

### 3.3 推理过程设计

**【推理过程写法】**：

```
推理 = 问题 → 假设 → 验证 → 结论

问题：
- 提出问题
- 明确推理目标

假设：
- 提出可能
- 列出可能选项

验证：
- 检验假设
- 排除/确认
- 发现新线索

结论：
- 得出结论
- 验证结论
- 揭示真相
```

**【避免的问题】**：
- 侦探"突然想到"
- 证据"突然发现"
- 推理缺乏过程

### 3.4 逻辑漏洞检查

**【常见漏洞】**：

| 漏洞类型 | 表现 | 修复 |
|---------|------|------|
| 机械降神 | 最后用"唯一没出场的人"做凶手 | 提前给出所有嫌疑人 |
| 证据孤证 | 单个证据定罪 | 多证据链 |
| 可能性唯一 | 推理是唯一的可能 | 增加其他可能再排除 |
| 条件遗漏 | 推理依赖未给出的条件 | 检查所有条件 |
| 时间漏洞 | 时间线矛盾 | 制作时间线表 |

## 四、诡计设计

### 4.1 经典诡计类型

| 诡计类型 | 描述 | 代表作 |
|---------|------|-------|
| **密室杀人** | 作案现场无法进入/离开 | 《钟表馆事件》 |
| **不在场证明** | 时间线造假 | 《占星术杀人魔法》 |
| **死亡信息** | 死亡信息有隐藏含义 | 《恶意》 |
| **雪地利器** | 利用自然条件 | 《雪满梁园》 |
| **分尸** | 尸体无法辨认 | 《解体诸因》 |
| **毒药** | 利用毒药特性 | 《毒巧克力案》 |

### 4.2 诡计设计方法

**【诡计设计公式】**：

```
诡计 = 目标 + 手法 + 反制 + 代价

目标：
- 要达成什么
- 迷惑谁

手法：
- 具体怎么做
- 利用什么条件

反制：
- 如何被识破
- 识破的关键

代价：
- 诡计的局限
- 凶手的风险
```

**【诡计复杂度控制】**：

```
诡计复杂度 = 读者理解力 × 阅读时间

简单诡计（1-2个手法）：
- 适合短篇
- 适合新人
- 容易设计

中等诡计（2-3个手法）：
- 适合中篇
- 需要组合
- 有一定挑战

复杂诡计（3+个手法）：
- 适合长篇
- 需要精密设计
- 注意不要过度复杂
```

## 五、伏笔与误导

### 5.1 伏笔设计

**【伏笔在悬疑中的作用】**：

```
伏笔 = 信息隐藏 + 信息揭示 + 信息联系

信息隐藏：
- 隐藏关键信息
- 让读者忽略
- 埋下伏笔

信息揭示：
- 适时揭示
- 与谜题相关
- 推动推理

信息联系：
- 线索关联
- 形成推理链
- 最终指向真相
```

**【伏笔技巧】**：

| 技巧 | 写法 | 效果 |
|------|------|------|
| 正常化 | 把关键信息伪装成正常描述 | 让读者忽略 |
| 分散化 | 线索分散在多处 | 难以注意 |
| 重复化 | 关键信息多次出现 | 加深印象 |
| 反转型 | 正面描述隐藏真相 | 误导读者 |

### 5.2 误导设计

**【误导类型】**：

```
误导 = 转移注意 + 错误推理 + 合理排除

转移注意：
- 用无关信息转移焦点
- 用其他嫌疑人转移注意
- 用其他谜题转移注意

错误推理：
- 让读者做错误推理
- 引导读者走向误区
- 提供支持错误推理的"证据"

合理排除：
- 让错误嫌疑人看起来更可疑
- 让正确嫌疑人看起来清白
- 利用心理盲点
```

**【误导原则】**：
```
误导 = 合理 + 公平 + 有效

合理：
- 误导必须合理
- 符合人物和情节
- 不显得刻意

公平：
- 误导不等于欺骗
- 读者有机会发现真相
- 不能有唯一解

有效：
- 能成功误导大部分读者
- 误导有层次
- 最终解答出乎意料
```

## 六、角色设计

### 6.1 侦探角色设计

**【侦探类型】**：

| 类型 | 特点 | 代表 |
|------|------|------|
| 理性侦探 | 冷静、逻辑 | 福尔摩斯 |
| 感性侦探 | 直觉、感性 | 马普尔小姐 |
| 行动侦探 | 硬汉、搏斗 | 马洛 |
| 专业侦探 | 利用专业技能 | 汤川学 |
| 业余侦探 | 意外卷入 | 普通人的视角 |

**【侦探设计公式】**：

```
侦探 = 能力 + 局限 + 性格 + 动机

能力：
- 观察力
- 推理能力
- 专业技能
- 人脉资源

局限：
- 性格弱点
- 知识盲点
- 情感偏见

性格：
- 说话方式
- 行为习惯
- 价值观

动机：
- 为什么要破案
- 个人情感
- 职业责任
```

### 6.2 嫌疑人设计

**【嫌疑人设计原则】**：

```
嫌疑人 = 可能性 + 合理性 + 深度

可能性：
- 都有杀人机会
- 都有杀人动机
- 看起来都可疑

合理性：
- 动机合理
- 行为合理
- 性格合理

深度：
- 不只是"嫌疑人"
- 有自己的故事
- 有自己的挣扎
```

**【嫌疑人分配】**：

```
嫌疑人结构 = 核心嫌疑人 + 辅助嫌疑人 + 误导嫌疑人

核心嫌疑人（1-2个）：
- 真正的凶手
- 动机最强/最深

辅助嫌疑人（2-3个）：
- 有嫌疑但不是凶手
- 推动调查
- 揭示动机

误导嫌疑人（1-2个）：
- 看起来最可疑
- 实际上是清白的
- 提供误导线索
```

### 6.3 凶手设计

**【凶手设计原则】**：

```
凶手 = 合理性 + 深度 + 意外性

合理性：
- 动机合理
- 手法可行
- 行为一致

深度：
- 凶手不只是"凶手"
- 有自己的故事
- 动机有社会/人性意义

意外性：
- 读者没猜到
- 但回看觉得合理
- 解答有"原来如此"感
```

## 七、悬疑节奏设计

### 7.1 开篇设计

**【开篇任务】**：

```
开篇 = 吸引注意 + 建立问题 + 展示氛围

吸引注意：
- 抛出谜题
- 制造悬念
- 引起好奇

建立问题：
- 核心问题是什么
- 关键人物是谁
- 期待什么

展示氛围：
- 确定类型基调
- 悬疑/恐怖/推理
- 建立节奏感
```

### 7.2 中段设计

**【中段任务】**：

```
中段 = 调查 + 推理 + 误导 + 深化

调查：
- 收集线索
- 访问证人
- 发现问题

推理：
- 提出假设
- 验证推理
- 排除/发现

误导：
- 设置误导
- 深化悬念
- 制造张力

深化：
- 引入新问题
- 深化动机
- 挖掘人性
```

### 7.3 高潮与解答

**【高潮设计】**：

```
高潮 = 临界点 + 揭示 + 震撼

临界点：
- 压力最大
- 真相即将揭晓
- 读者期待最高

揭示：
- 真相大白
- 逻辑完整
- 情感释放

震撼：
- 意外性
- 合理性
- 深度
```

## 八、常见问题与避坑

### 问题1：机械降神

**表现**：最后用没出场的人做凶手

**避坑**：
- 提前给出所有嫌疑人
- 凶手必须在读者视野内
- 不能临时添加线索

### 问题2：线索不公平

**表现**：作者知道但没给读者线索

**避坑**：
- 核心线索必须提前给出
- 列出所有线索清单
- 验证读者能否推理

### 问题3：反转为反转

**表现**：没有意义的反转

**避坑**：
- 反转必须有深度
- 反转必须合理
- 反转必须推动主题

### 问题4：推理过程缺失

**表现**：侦探"突然想到"

**避坑**：
- 展示推理过程
- 给出推理依据
- 让读者跟上思路

### 问题5：动机牵强

**表现**：凶手动机不合理

**避坑**：
- 动机必须合理
- 动机必须有深度
- 动机最好有社会意义

## 九、标杆作品分析

### 9.1 《白夜行》分析

**为什么好**：
- 社会派推理典范
- 双主线结构
- 动机深刻
- 结局震撼

- 诡计设计精妙
- 动机深刻
- 逻辑严密
- 情感震撼
- 反转合理

**学什么**：
- 如何设计精妙诡计
- 如何写"献身"式动机
- 如何平衡逻辑与情感
- 如何写反转

- 孤岛模式经典

- 结局意外

**学什么**：

- 如何设计节奏
- 如何创新模式

## 十二、评分体系对应

### 12.1 悬疑/推理类权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 基础表达 | 10% | 文笔服务节奏 |
| 世界观与设定 | 5% | 设定非核心 |
| 剧情与结构 | **40%** | 结构决定推理质量 |
| 节奏与氛围 | 20% | 悬疑氛围至关重要 |
