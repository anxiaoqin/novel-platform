# 审核与自检清单

> 本指南是基于评分体系的自检工具，帮助快速定位问题并修复。

---

## 一、使用说明

### 1.1 清单结构

本清单按评分体系的五大维度组织，每个维度包含：
- 快速判断项（是/否）
- 问题识别
- 修复优先级

### 1.2 使用方法

```
第一步：通读全文，记录直觉感受

第二步：按维度逐项检查

第三步：统计问题数量和严重程度

第四步：确定修改优先级

第五步：针对问题查阅对应指南
```

## 九、修改优先级判断

### 9.1 优先级判断标准

| 优先级 | 问题类型 | 说明 |
|--------|---------|------|
| **P0（必须修复）** | 致命问题 | 严重影响阅读，如逻辑矛盾、视角越界 |
| **P1（应该修复）** | 严重问题 | 明显影响质量，如张力不足、对话无区分 |
| **P2（建议修复）** | 一般问题 | 影响阅读体验，如用词重复、修辞堆砌 |
| **P3（可选修复）** | 优化项 | 提升质量，不修复也可以 |

| 问题 | 优先级 | 原因 |
|------|--------|------|

| 视角越界 | P0 | 让读者出戏 |
| 战力崩坏 | P1 | 影响爽感 |
| 无钩子（网文） | P1 | 影响追更 |
| 张力不足 | P1 | 影响阅读 |
| 对话无区分 | P1 | 影响角色 |
| 伏笔遗忘 | P1 | 影响结构 |
| 金手指过强 | P2 | 影响挑战 |
| 用词重复 | P2 | 影响阅读 |
| 修辞堆砌 | P3 | 影响文风 |
