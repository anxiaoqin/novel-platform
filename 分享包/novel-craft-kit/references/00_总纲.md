# AI小说创作指南·总纲

> 本指南是《小说评分审核体系》的创作转化版——评分体系告诉你"审什么"，本指南教你"怎么写"。

---

## 一、指南定位与使用方法

### 1.1 指南定位

本指南的核心目标是：**让AI拿到这些文档后，能够稳定、有效、高质量地输出或优化小说内容。**

- **不是评分标准**：评分体系是裁判，本指南是教练
- **不是写作理论**：跳过"为什么"，直接给出"怎么做"
- **是操作手册**：每个方法都附有AI可执行的prompt模板

### 1.2 文档结构


## 二、AI创作的核心原则

### 2.1 AI与人类创作的差异

| 维度 | 人类创作 | AI创作 |
|------|---------|--------|
| **优势** | 情感深度、直觉判断、风格独特性 | 信息整合、速度、稳定性、可复制性 |
| **劣势** | 疲劳、情绪波动、一致性难保证 | 容易套路化、缺乏真正的"惊喜"、细节容易重复 |
| **擅长** | 原创性突破、情感共鸣、文学性 | 结构完整、规则遵守、多样化生成 |

### 2.2 AI创作的铁律

#### 铁律一：设定必须先于写作

**错误示范**：
```
请写一个玄幻小说
```

**正确示范**：
```
请写一个玄幻小说：
- 修炼体系：炼气→筑基→金丹→元婴，每级有三重小境界
- 世界观：修真界与世俗界分离，修士不得干预凡人
- 核心冲突：末法时代来临，灵气枯竭
- 主线：主角发现可以通过吸收雷电修炼，踏上逆天之路
```

#### 铁律二：每个角色必须有"禁止清单"

AI容易在长文本中"忘记"角色限制。必须在prompt中明确：

```
角色设定：
- 姓名：林远
- 性格：沉默寡言，重情重义
- 说话风格：简短，句末常带"……"表示犹豫
- 【禁止行为】：不使用"突然暴怒""仰天长啸"等过度反应
- 【禁止能力】：不能瞬移、不能预知未来
```

#### 铁律三：长文本必须分段生成并回顾

AI在长文本中容易出现：
- 前后矛盾
- 角色OOC（Out of Character）
- 战力崩坏

**应对方法**：每3000字进行一次"剧情回顾"，让AI总结当前状态。

### 2.3 AI擅长的写作任务

| 任务类型 | AI表现 | 原因 |
|---------|--------|------|
| 场景描写 | ⭐⭐⭐⭐⭐ | 可通过五感法稳定输出 |
| 对话生成 | ⭐⭐⭐⭐ | 需要给定角色性格 |
| 设定扩展 | ⭐⭐⭐⭐⭐ | 信息整合能力强 |
| 情节填充 | ⭐⭐⭐ | 需要明确主线 |
| 情感深化 | ⭐⭐ | 需要人类引导 |
| 风格创新 | ⭐⭐ | 容易套路化 |

### 2.4 AI容易犯的错误

| 错误类型 | 具体表现 | 修复方法 |
|---------|---------|---------|
| 套路化 | 频繁使用"突然""猛然""霎时间" | 添加"避免陈词滥调"指令 |
| 重复 | 相同形容词反复出现 | 指定用词清单 |
| 越界 | 第三人称写内心独白 | 明确视角限制 |
| 告诉 | "他很悲伤"代替情感展示 | 强调"展示法"要求 |
| 空转 | 无意义描写填充字数 | 指定信息密度要求 |

## 四、AI创作的"铁律清单"

以下规则适用于所有类型、所有阶段的AI创作，违反任一条都可能导致输出质量大幅下降：

### 4.1 创作前必须确认的信息

- [ ] 小说类型（科幻/奇幻/玄幻/都市/悬疑/儿童/网文）
- [ ] 篇幅（短篇≤2万字/中篇2-8万字/长篇>8万字）
- [ ] 叙事视角（第一人称/第三人称/全知）
- [ ] 核心设定（世界观、能力体系、核心冲突）
- [ ] 主线剧情（起点事件、核心冲突、目标）

### 4.2 写作过程中必须遵守的规则

- [ ] 每个场景必须有明确目的（推动剧情/揭示性格/营造氛围）
- [ ] 每个对话必须有潜台词或信息传递
- [ ] 每段描写必须服务情感或剧情
- [ ] 禁止连续三段使用相同的高频词
- [ ] 禁止用"他很XX"代替情感展示
- [ ] 禁止越界描写（写视角人物不可能知道的内容）

### 4.3 长篇创作必须执行的流程

- [ ] 每3000字进行一次剧情回顾
- [ ] 每10000字进行一次战力/能力核查
- [ ] 每50000字进行一次伏笔梳理
- [ ] 章节间必须有过渡或钩子

## 五、AI指令模板通用格式

### 5.1 标准创作指令模板

```
【创作任务】
[具体写作任务描述]

【背景设定】
- 类型：[小说类型]
- 视角：[叙事视角]
- 世界观：[简要描述]
- 当前场景：[场景描述]

【角色信息】
[列出当前场景涉及的角色及其关键设定]

【情节要求】
[必须包含的情节要素]
[必须避免的情节要素]

【风格要求】
- 语调：[如紧张/压抑/欢快/悬疑]
- 节奏：[如快节奏短句/慢节奏长句]
- 信息密度：[如每段至少包含一个信息点]

【输出要求】
- 字数：[目标字数]
- 格式：[如场景描写+对话混合]
```

### 5.2 优化指令模板

```
【待优化内容】
[原文内容粘贴]

【优化目标】
[如：增强紧张感/提升信息密度/增强角色性格]

【具体要求】
[如：使用更多动词/减少副词使用/增加潜台词]

【禁止项】
[明确列出不希望出现的内容]
```
