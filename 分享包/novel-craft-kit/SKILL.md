# Novel Craft Kit · 小说创作评分与指南工具包

> 一套面向AI辅助小说创作的完整工具包，包含评分审核体系（v2.0）和AI创作指南（15个文件）。加载后AI可据此对小说进行多维度评分、审核和创作指导。

---

## 技能简介

本工具包提供两大核心能力：

1. **评分审核**：6大维度（基础表达/世界观与设定/剧情与结构/节奏与氛围/角色与人物/AI痕迹检测）× 30+子项的百分制评分体系，含类型权重适配、写作红线审核、AI修正系数
2. **创作指导**：从世界观构建到文字打磨的全流程操作手册，覆盖7种小说类型（科幻/奇幻玄幻/都市/悬疑推理/儿童文学/网文/纯文学）

### 适用场景

- 对已完成的小说作品进行多维度评分审核
- 指导AI进行高质量小说创作或改写优化
- "去AI化"改写：检测AI痕迹并针对性优化
- 写作红线自检：确保作品不踩常见创作陷阱

### 不适用场景

- 诗歌、散文、剧本等非叙事类文体
- 仅需简单文字润色的轻量需求

## 使用方法

### 场景一：评分审核

```
1. 确定作品类型和篇幅 → 查阅 references/评分审核体系.md 第七部分的权重表
2. 通读全文 → 建立整体印象
3. 逐维度评分 → 按 references/评分审核体系.md 第二到第六部分的子项标准逐项打分
4. AI痕迹检测 → 调用 ai-text-detector 技能检测AI率，查修正系数表
5. 权重汇总 → 计算加权总分 + AI修正系数
6. 出具报告 → 按 references/评分审核体系.md 第八部分的报告模板
```

### 场景二：创作指导

```
1. 明确类型 → 读对应 references/AI创作指南/08-13 类型专项文档
2. 构建世界观 → 读 references/AI创作指南/02_世界观设定.md
3. 设计角色 → 读 references/AI创作指南/03_角色设计.md
4. 规划剧情 → 读 references/AI创作指南/01_主线与支线设计.md
5. 开始写作 → 读 references/AI创作指南/05-07 基础写作指导
6. 自检优化 → 读 references/AI创作指南/14_审核与自检清单.md
```

### 场景三：去AI化改写

```
1. 修改前检测 → 调用 ai-text-detector 获取各段落AI率
2. 标记高AI率段落 → AI率>30%的段落
3. 针对性改写 → 按 references/评分审核体系.md 第六点五部分的4项指标优化方向
4. 修改后检测 → 重新检测对比
5. 验收 → AI率降至目标范围（硬科幻<20%/纯文学<10%/网文<30%）
```
