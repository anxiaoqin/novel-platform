## 📋 创作规范速查

### 视角规则
- **1-5卷**：男主第一人称主线 + 林清雪第三人称副线（双线并行）
- **第6卷起**：统一男主第一人称

### 文风要求
- 口语化吐槽、网络热梗不断
- 轻松搞笑 + 硬核热血双线兼顾
- 每章按七段爆款结构写
- 卷末必留抓人强钩子

### 战力设定
- 男主全程后天苦修无敌
- 跨世界战力直接清零
- 无无脑天生金手指
