## 一、作品基本信息

| 项目 | 内容 |
|------|------|
| 作品名称 | 两个星球 |
| 作品类型 | 硬科幻短篇（双文明接触题材） |
| 预估篇幅 | 约8,000-10,000字 |
| 核心设定 | 双星系统中星蓝星（视觉/声波文明）与橘红星（振动/共振文明）的第一次"真正"接触，因解码器预设导致两百年误解 |

| 叙事视角 | 第三人称限制视角，多角色轮换 |
| 时间结构 | 倒计时框架（72小时→14天交汇期） |

## 二、权重说明

### 硬科幻短篇权重配置

| 维度 | 权重 | 说明 |
|------|------|------|
| **基础表达** | **15%** | 文字质量、句式、视角、对话等底层能力 |
| **世界观与设定** | **35%** | 硬科幻核心：设定自洽性、技术可信度、创新度 |
| **剧情与结构** | **25%** | 起承转合、转折合理性、伏笔回收、情感弧线 |
| **节奏与氛围** | **10%** | 张力曲线、信息投放、留白控制 |
| **角色与人物** | **15%** | 主角辨识度、配角功能、反派质量 |
| **AI痕迹检测** | 修正系数 | 通过本地算法估算AI率，给出修正建议 |

**权重选择理由**：
- 硬科幻核心是"设定"（世界观权重35%最高）
- 短篇篇幅限制剧情展开空间（剧情权重25%）
- 基础表达是硬科幻文字"可信度"的保障（权重15%）
- 角色服务于设定与剧情（权重15%）
- 节奏氛围锦上添花（权重10%）
