# 写作红线：两个星球

## 创作铁律

### 继承自《四维教室》儿童版

1. **误读是核心**：任何场景都不能"绕开"误读直接达成理解
   - 绝对不允许：一方直接"学会"了对方的语言，然后翻译
   - 正确方式：碰撞/共振/物理交互产生"意外的理解"

2. **感知即存在**：每个文明的感知方式必须决定"什么是真实的"
   - 星蓝星：视觉模式主导，声波是次要信息
   - 橘红星：振动模式主导，电磁感知是次要信息
   - 绝对不允许：任何一方"想象"对方的世界

3. **善意被扭曲**：任何"好的意图"都必须被对方误解
   - 绝对不允许：一方发出善意，对方直接接收善意
   - 正确方式：善意到达对方时，变形为威胁或漠视

### 成人版新增铁律

4. **代价不可撤销**：任何接触/碰撞必须有真实的、不可逆的代价
   - 林的死亡是真实的
   - 晶体的崩解是真实的
   - 这些代价不能被"时间治愈"或"后来弥补"

5. **恐惧是理性的**：每个角色的恐惧必须有"合理的逻辑"
   - 绝对不允许：将恐惧写成"非理性"或"愚蠢"
   - 正确方式：每个角色的恐惧都基于他们的历史创伤和制度压力

6. **理解不等于和解**：知道对方在想什么，不等于接受或原谅
   - 绝对不允许：理解发生后，双方"和解"或"结盟"
   - 正确方式：理解发生后，每个角色面临更艰难的选择

## 绝对不能写的东西

### 内容红线

| 禁止项 | 原因 | 替代方案 |
|--------|------|----------|
| 任何形式的"大团圆结局" | 违背成人向定位 | 悬置、开放性选择 |
| "反派"角色 | 违背"恐惧是理性的"原则 | 有合理恐惧的对手 |
| 英雄式的自我牺牲换来和解 | 违背"代价不可撤销" | 牺牲后，选择仍在 |
| 单方面的"觉醒"或"转变" | 违背"复杂性"要求 | 每个角色都有未解决的问题 |
| 解释性的内心独白超过200字 | 违背节奏要求 | 通过行为和对话展示 |
| 直接描述对方的"情感" | 违背感知差异原则 | 通过振动/声波模式的物理描述间接展示 |
| "他们其实是一样的"桥段 | 违背"真正的差异"原则 | 两边的差异保留，只是更理解了 |

### 叙事红线

| 禁止项 | 原因 | 替代方案 |
|--------|------|----------|
| "后来"的叙述者补充 | 违背实时叙事原则 | 所有信息必须通过角色实时体验 |
| 全知视角的道德判断 | 违背开放性主题 | 让读者自己判断 |
| 总结性的"主题陈述" | 违背"不告知"原则 | 主题通过情节自然呈现 |
| "原来一切都是误会"桥段 | 违背"误解不需要恶意"原则 | 误解持续，理解在别处发生 |

### 科幻硬约束

| 禁止项 | 原因 | 正确做法 |
|--------|------|----------|
| 违反物理定律没有解释 | 违背硬科幻定位 | 所有技术设定必须有物理学基础 |
| "心灵感应"或"即时通讯"跨越光年 | 违背相对论约束 | 交汇期=通讯窗口，延迟是4分钟到4小时 |
| 任意改变感知方式 | 违背感知差异原则 | 星蓝星永远视觉优先，橘红星永远振动优先 |
| "高级文明"帮助解决冲突 | 违背自主原则 | 两边都犯错，都承担后果 |

## 基调守则

### 必须保持的基调元素

1. **冷硬的精确**：不用形容词堆砌，用精确的物理描述
   - ✅ "频率从200Hz跳至15kHz，晶体的共振响应曲线显示......"
   - ❌ "一道可怕的橙红色光芒席卷了一切"

2. **沉默的重量**：很多意义在对话之外
   - 场景中至少20%应该是"无对话"的纯粹描写或行为展示
   - 最强的交流发生在通讯中断之后

3. **疼痛的可触及性**：伤亡必须"可触及"
   - 林的死亡不是"数字"，是苏薇的"具体记忆"
   - 晶体崩解不是"损失"，是共振者-回声的"家族记忆"

4. **选择的悬置**：永远不给出"正确答案"
   - 每个选择都有代价
   - 故事的终点是问题，不是答案

### 必须避免的基调元素

| 避免 | 原因 | 替代 |
|------|------|------|
| 煽情化的悲壮 | 违背"冷硬精确" | 克制但有力量的悲怆 |
| 浪漫化的异星接触 | 违背"成人向"定位 | 笨拙的、误解的、痛苦的理解 |
| 科幻设定的自我崇拜 | 违背"服务于主题" | 技术细节简洁，为情感让路 |
| 过度解释设定 | 违背"展示优于告知" | 让读者在情节中推断世界观 |

## 节奏约束

### 各节节奏要求

| 节 | 节奏要求 | 禁止节奏 |
|----|----------|----------|
| 第一节 | 稳健铺垫，信息密集 | 拖沓的日常描写 |
| 第二节 | 加速，紧张上升 | 过多内心独白 |
| 第三节 | 高张力，对话密集 | 长篇技术解释 |
| 第四节 | 爆发、停顿、再爆发 | 平滑过渡 |
| 第五节 | 沉重、追问、深思 | 快速推进 |
| 第六节 | 密集对话、选择时刻 | 场景跳跃 |
| 第七节 | 缓慢、留白、开放 | 总结性陈述 |

### 字数约束

- **单段落**：最多300字
- **单场景对话**：最多1000字
- **内心独白**：最多200字，且必须通过行为触发
- **技术描写**：单次最多150字

### 节奏检查清单

- [ ] 每节至少有一个"张力峰值"
- [ ] 每3000字至少有一个"小悬念"
- [ ] 章节末尾必须有"钩子"（但不能是人为制造的悬念）
- [ ] 碰撞场景（第四节）必须保持高节奏，即使在描写痛苦时

## 科幻硬约束清单

### 通讯机制

- [ ] 交汇期才能通讯，窗口14地球日
- [ ] 通讯延迟从4分钟（近）到4小时（远）
- [ ] 星蓝星用声波+视觉，橘红星用振动+共振
- [ ] 误读的原因是"解码器预设"，不是"信息缺失"

### 物理机制

- [ ] 晶体的崩解基于压电效应的共振失稳
- [ ] 碰撞产生的"意外频率"必须是双方都未预设的
- [ ] 探测器的设计基于已知物理原理，任何幻想元素必须有"半硬科幻"解释

### 感知差异

- [ ] 星蓝星角色永远"先看到，再听到"
- [ ] 橘红星角色永远"先感受振动，再感知其他"
- [ ] 任何跨感知的"直接理解"必须被描述为"错觉"或"误解"

### 社会设定

- [ ] 防御协调署的权力基于"历史创伤的制度化"
- [ ] "噪音过滤"协议有晶体网络的物理根源
- [ ] 每个政治决策都必须追溯到"个人创伤+制度压力"

## 主题约束

### 必须探讨的主题

1. **"善意被扭曲"的多层含义**
   - 文本层面：声波被振动翻译器误读
   - 制度层面：问候被安全协议误判
   - 存在论层面：存在方式本身决定什么是"可理解的"

2. **"恐惧是理性的"，但不等于"正确"**
   - 每个角色的恐惧都有逻辑
   - 但逻辑不等于真相
   - 承认恐惧合理，不等于接受恐惧的后果

3. **"理解"的代价**
   - 理解需要碰撞
   - 碰撞造成伤亡
   - 伤亡不能撤销
   - 这是理解本身的问题，还是我们的方式的问题？

4. **"我是谁"的危机**
   - 苏薇：我是导师的学生还是背叛者？
   - 卡恩：我是保护者还是恐惧的囚徒？
   - 林：我是母亲的延续还是独立的个体？
   - 共振者-回声：我是父辈记忆的守护者还是背叛者？
   - 守卫者-沉默：我是保护者还是狱卒？

### 禁止的主题简化

| 禁止简化 | 原因 | 保持复杂 |
|---------|------|----------|
| "只要沟通就能理解" | 违背核心设定 | 沟通本身可能是误解的来源 |
| "恐惧是进步的阻碍" | 违背恐惧的合理性 | 恐惧保护了什么，也囚禁了什么 |
| "科学可以解决一切" | 违背认知局限性 | 科学本身就是一种感知方式 |
| "和解是最终答案" | 违背开放性 | 和解之后是什么？ |

## 最终检查清单

### 写作前的检查

- [ ] 每个角色的"核心选择"已明确
- [ ] 每个选择都有"两个都痛苦"的选项
- [ ] 每个误读都有物理/化学/感知的原因
- [ ] 每个伤亡都是"不可撤销的"

### 写作中的检查

- [ ] 没有超过300字的单段落
- [ ] 没有超过200字的内心独白
- [ ] 对话中通过行为展示动机，而非直接陈述
- [ ] 20%的场景是无对话的描写

### 写作后的检查

- [ ] 每个"真相揭示"都有"新的问题"伴随
- [ ] 结尾是"问题"而非"答案"
- [ ] 读者在合上故事后，有"想继续思考"的空间
- [ ] 没有任何角色因为"理解了"而变得完整或满足

## 成人向核心提醒

> **"两个星球"的核心不是"两个文明终于理解了彼此"。**
>
> **它的核心是："理解了之后，你愿意接受吗？"**
>
> **理解不等于接受。接受不等于改变。改变不等于和解。**
>
> **这个故事要问的是：知道对方在想什么，你的生活变得更简单了吗？**
>
> **答案是：不。更难了。**
>
> **但更难，不等于不值得。**
