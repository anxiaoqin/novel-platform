# 类型专项·都市小说

> 本指南针对都市小说的特有创作要点。

---

## 一、都市小说核心原则

1. **现实感是生命**：让读者相信这是真实的世界
2. **角色是核心**：人物是都市小说的最大吸引力
3. **情感是引擎**：情感纠葛驱动阅读
4. **社会是底色**：时代特征、行业生态
5. **细节是质感**：真实的细节创造沉浸感

---

## 二、都市小说创作要点

### 2.1 现实感构建

**【现实感三要素】**：

```
现实感 = 生活细节 + 行业真实 + 社会逻辑

生活细节：
- 居住环境
- 日常消费
- 交通出行
- 社交方式

行业真实：
- 行业规则
- 职业特点
- 从业者心态
- 行业黑话

社会逻辑：
- 阶层逻辑
- 人际关系
- 资源分配
- 规则与潜规则
```

**【避免的问题】**：
- 月薪三千住市中心豪宅
- 职场描写与现实脱节
- 社会阶层与生活不符
- 行业细节明显错误

### 2.2 行业描写深度

**【行业描写的层次】**：

```
行业描写 = 表层 + 中层 + 深层

表层（所有人都知道）：
- 基本工作内容
- 外在形象
- 常见场景

中层（从业者知道）：
- 行业规则
- 潜规则
- 内部逻辑
- 晋升路径

深层（精英才知道）：
- 行业内幕
- 利益格局
- 权力游戏
- 未来趋势
```

**【写作方法】**：
```
行业展示公式：
主角经历 → 揭示行业 → 展示细节

通过主角的视角和经历，逐步展示行业的不同层次。
不是"说明书"，而是"体验"。
```

**【AI指令模板】**：

```
请设计行业描写：

【行业名称】
[如：互联网、医疗、金融、法律]

【行业层次】
表层：
- 工作内容：[描述]
- 外在形象：[描述]

中层：
- 行业规则：[列出3-5条]
- 潜规则：[列出2-3条]

深层：
- 利益格局：[描述]
- 权力结构：[描述]

【主角与行业】
- 主角职位：[描述]
- 主角视角：[资深/新人/边缘]
- 展示方式：[通过经历展示]

【细节清单】
- 典型场景：[列出3个]
- 行业术语：[列出]
- 标志性物件：[列出]
```

### 2.3 社会阶层描写

**【阶层描写要点】**：

```
阶层描写 = 物质差异 + 精神差异 + 行为差异 + 关系差异

物质差异：
- 收入与消费
- 居住环境
- 交通方式
- 教育资源

精神差异：
- 价值观
- 焦虑点
- 追求目标
- 视野局限

行为差异：
- 说话方式
- 时间观念
- 社交模式
- 决策方式

关系差异：
- 亲情模式
- 友情模式
- 爱情模式
```

### 2.4 都市情感描写

**【都市情感特点】**：

```
都市情感 = 现实压力 + 独立追求 + 情感孤独 + 阶层差异

现实压力：
- 工作与生活平衡
- 经济压力
- 房价焦虑
- 职业发展

独立追求：
- 个人价值
- 自我实现
- 不愿妥协

情感孤独：
- 城市孤独感
- 熟人社会的消失
- 真情难觅

阶层差异：
- 门当户对
- 圈层固化
- 跨越的困难
```

## 三、都市小说角色设计

### 3.1 都市主角设计

**【都市主角类型】**：

| 类型 | 特点 | 代表作品 |
|------|------|---------|
| 职场新人 | 成长型，面对职场挑战 | 《杜拉拉升职记》 |
| 中产阶层 | 危机型，面对阶层滑落 | 《都挺好》 |
| 创业者 | 冒险型，追求自我价值 | 《大江大河》 |
| 普通市民 | 现实型，面对生活压力 | 《贫嘴张大民的幸福生活》 |

**【主角设计要素】**：

```
都市主角 = 职业身份 + 阶层位置 + 性格特点 + 情感状态

职业身份：
- 职位
- 行业
- 资历

阶层位置：
- 家庭背景
- 经济实力
- 社会资源

性格特点：
- 优点
- 缺点
- 核心矛盾

情感状态：
- 感情状况
- 情感需求
- 情感障碍
```

### 3.2 都市配角设计

**【都市配角类型】**：

```
都市配角 = 功能定位 + 阶层属性 + 性格特征

功能定位：
- 闺蜜/兄弟：情感支持
- 竞争对手：事业推动
- 上司/下属：职场关系
- 前任/追求者：情感纠葛
- 父母/亲戚：原生家庭

阶层属性：
- 与主角的阶层关系
- 阶层差异如何制造冲突

性格特征：
- 配合功能
- 避免脸谱化
```

### 3.3 情感关系设计

**【都市情感关系类型】**：

| 关系类型 | 特点 | 冲突来源 |
|---------|------|---------|
| 办公室恋情 | 职场+情感双重关系 | 利益冲突、公开与否 |
| 相亲对象 | 现实压力下的选择 | 条件vs感觉 |
| 门第差异 | 阶层差距 | 家庭反对、价值观差异 |
| 初恋/前任 | 情感阴影 | 过去的阴影、现实对比 |
| 闪婚/裸婚 | 极端选择 | 经济压力、家庭压力 |

**【情感线设计公式】**：

```
情感发展 = 吸引 → 误解/障碍 → 危机 → 突破/决裂

吸引：
- 如何相遇
- 什么吸引
- 吸引的独特性

误解/障碍：
- 什么阻碍
- 如何误解
- 现实vs情感

危机：
- 什么考验
- 如何选择
- 代价是什么

突破/决裂：
- 结果如何
- 成长还是退步
- 意义是什么
```

## 四、都市小说世界观

### 4.1 城市描写

**【城市描写要素】**：

```
城市 = 地理 + 空间 + 符号 + 氛围

地理：
- 城市定位
- 区域划分
- 地标建筑

空间：
- 物理空间
- 社会空间
- 阶层空间

符号：
- 城市标签
- 文化符号
- 阶层符号

氛围：
- 城市节奏
- 人群特征
- 生活气息
```

**【避免】**：
- 城市只是背景板
- 缺乏地域特色
- 城市与人物脱节

### 4.2 时代特征

**【时代特征的融入】**：

```
时代 = 背景 + 元素 + 影响

背景：
- 时代背景
- 社会环境
- 发展趋势

元素：
- 时代符号
- 流行文化
- 热点事件

影响：
- 对人的影响
- 对关系的影响
- 对价值观的影响
```

### 4.3 社会生态

**【社会生态描写】**：

```
社会生态 = 阶层 + 圈层 + 规则 + 潜规则

阶层：
- 上中下阶层
- 阶层流动
- 阶层固化

圈层：
- 职业圈
- 社交圈
- 兴趣圈

规则：
- 明的规则
- 暗的规则

潜规则：
- 行业潜规则
- 社会潜规则
```

## 五、都市小说叙事技巧

### 5.1 日常生活叙事

**【日常叙事要点】**：

```
日常叙事 = 场景选择 + 细节捕捉 + 意义提炼

场景选择：
- 选择有代表性的日常场景
- 场景要能反映人物和主题
- 避免流水账

细节捕捉：
- 有意味的细节
- 反映阶层的细节
- 推动情感的细节

意义提炼：
- 从日常中见深刻
- 以小见大
- 日常即意义
```

### 5.2 对话与潜台词

**【都市对话特点】**：

```
都市对话 = 身份感 + 阶层感 + 潜台词

身份感：
- 职业用语
- 阶层用语
- 圈层用语

阶层感：
- 说话方式差异
- 内容差异
- 态度差异

潜台词：
- 不说的话比说的话更重要
- 阶层差异导致的潜台词
- 都市人的"不说"
```

### 5.3 情感节奏

**【都市情感节奏】**：

```
情感节奏 = 压抑 → 裂缝 → 爆发 → 沉淀

压抑：
- 现实压力
- 不说出口
- 积累

裂缝：
- 矛盾出现
- 第一次冲突
- 关系的动摇

爆发：
- 核心矛盾
- 关键选择
- 情感决堤

沉淀：
- 反思
- 成长
- 新的平衡
```

## 六、都市小说常见问题与避坑

### 问题1：悬浮感

**表现**：生活不真实，读者出戏

**避坑**：
- 深入了解所写行业
- 注意阶层差异
- 真实细节替代想象

### 问题2：脸谱化

**表现**：霸道总裁、傻白甜套路

**避坑**：
- 角色要有复杂性
- 避免标签化
- 增加缺陷和弱点

### 问题3：没有都市味

**表现**：换个背景就是古代

**避坑**：
- 融入时代特征
- 体现城市特色
- 反映都市情感

### 问题4：情感狗血

**表现**：误会、癌症、车祸老套路

**避坑**：
- 真实情感冲突
- 现实问题驱动
- 细腻情感描写

### 问题5：说教化

**表现**：通过角色说教

**避坑**：
- 用行动和选择代替说教
- 让读者自己得出结论
- 主题融入故事

## 七、标杆作品分析

### 7.1 《都挺好》分析

**为什么好**：
- 家庭关系真实
- 社会议题精准
- 角色立体
- 情感细腻
- 现实意义强

**学什么**：
- 如何写原生家庭
- 如何平衡多个角色
- 如何用情节推动议题
- 如何写现实主义

### 7.2 《人民的名义》分析

**为什么好**：
- 行业真实
- 尺度大胆
- 人物复杂
- 情节紧张
- 社会影响大

**学什么**：
- 如何写敏感题材
- 如何平衡可读性和严肃性
- 如何写政治生态
- 如何写复杂人物
