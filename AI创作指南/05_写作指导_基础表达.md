# 写作指导·基础表达

> 本指南对应评分体系"基础表达"维度，包含：用词、句式、修辞、视角、对话、信息密度、展示vs告知。

---

## 一、核心原则

1. **精准是底线**：每个词都要用对，不能词不达意
2. **节奏是呼吸**：长短句搭配让阅读有呼吸感
3. **克制是高级**：修辞要用在刀刃上，而不是堆砌
4. **视角是窗口**：读者只能从视角窗口看世界
5. **对话即性格**：对话是角色塑造的重要工具
6. **信息即价值**：每句话都应该有信息量

## 二、用词精准法

### 2.1 动词选择法

**【目标】**：让动词有力量感，避免弱动词

**【方法】**：强动词替代弱动词

| 弱动词 | 强替代 | 适用场景 |
|--------|--------|---------|
| 是、有、做了 | 变成、存在、完成 | 尽量避免"是"字句 |
| 走、跑、说 | 踏入、疾冲、低语 | 动作描写 |
| 看、想、觉得 | 凝视、思索、感悟 | 心理描写 |
| 突然、猛然 | 一瞬间、刹那间 | 少用，可替换 |

**【正例特征】**：
- "他蜷缩在角落"（蜷缩 > 蹲）
- "脚步声逼近"（逼近 > 传来）
- "她的目光钉在他身上"（钉 > 看）

**【反例特征】**：
- "他很生气" → 改用肢体语言
- "他很快地跑" → 改用更有力量的动词

**【AI指令模板】**：

```
请优化以下段落的用词：

【原文】
[粘贴原文]

【优化要求】
1. 找出弱动词并替换
2. 找出重复用词并替换
3. 用具体动作替代情绪描述

【输出格式】
- 原文用词分析：[列出问题]
- 优化后段落：[写出优化结果]
- 替换对照表：[弱词→强词]
```

### 2.2 形容词克制法

**【目标】**：少用形容词，用具体细节替代

**【方法】**：形容词替代公式

```
形容词 → 细节展示

"红色的花" → "盛开的玫瑰"
"很高的楼" → "三十层的写字楼"
"很可怕的声音" → "像指甲划过玻璃的声音"
```

**【检查原则】**：
- 这个形容词是否可以用具体事物替代？
- 这个形容词传达的信息是否可以通过感官呈现？

**【AI指令模板】**：

```

【原文】
[粘贴]

【要求】
- 用具体事物替代抽象形容词
- 用感官描写替代情绪形容词
- 用动词替代状态形容词

【输出】
优化后：[写出结果]
```

### 2.3 冗余词识别与删除

**【目标】**：删除不影响意思的词

**【常见冗余类型】**：

| 冗余类型 | 例句 | 修正 |
|---------|------|------|
| 同义重复 | "他的眼睛" | "他" |
| 修饰重复 | "黑色的乌鸦" | "乌鸦" |
| 明显事实 | "他用手拿起东西" | "他拿起" |
| 连续副词 | "非常很极其" | "极其" |

**【AI指令模板】**：

```

【输出】

```

## 三、句式节奏法

### 3.1 长短句搭配法

**【目标】**：让阅读有节奏感

**【方法】**：情绪-句式对应

```
句式节奏公式：
情绪紧张 → 短句
情绪舒缓 → 长句
情绪激烈 → 短句+感叹
情绪沉思 → 长句+停顿

节奏模板：
紧张场景：[短短短短长][短短短短长]
舒缓场景：[长长长长长][长长长短]
高潮场景：[短短短短短][短短短短短]
```

**【正例特征】**：
- 紧张：刀锋划过。血滴落。寂静。
- 舒缓：他缓缓睁开眼睛，阳光从窗帘缝隙透进来。

**【AI指令模板】**：

```
请调整以下段落的句式节奏：

【原文】
[粘贴]

【目标情绪】
[紧张/舒缓/激烈/沉思]

【要求】
- 短句不超过10字
- 长句不超过30字
- 根据情绪调整长短比例

【输出】
节奏分析：[分析当前节奏]
优化后：[写出结果]
```

### 3.2 情绪-句式对应法

**【不同情绪的句式建议】**：

| 情绪 | 句式特征 | 举例 |
|------|---------|------|
| 紧张 | 短促、断裂、省略 | "跑！快！" |
| 愤怒 | 短促、感叹、重复 | "他不会放过你！不会！" |
| 悲伤 | 断断续续、长停顿 | "她走了……永远地……" |
| 思考 | 长句、复杂、从句 | "他开始思考，如果当初做了不同的选择，现在会怎样" |
| 兴奋 | 短促、感叹、省略 | "赢了！我们赢了！" |
| 平静 | 长短交替、平稳 | "阳光很好，他坐在窗边，想着过去" |

## 四、修辞使用法

### 4.1 何时用比喻

**【目标】**：用比喻创造"陌生化"效果

**【方法】**：比喻设计公式

```
好比喻 = 本体 + 喻体 + 新鲜联系

公式：
不是[常见喻体]，而是[意想不到但合理的喻体]

"她的眼睛" 
→ 不是"像星星" 
→ 而是"像手术刀，冷而精准"
```

**【正例特征】**：
- "她的眼睛像手术刀"（新鲜）
- "爱情像战争"（对比产生张力）
- "时间像沙漏"（具象化抽象）

**【反例特征】**：
- "眼睛像星星"
- "心像刀割"
- "泪如雨下"

**【AI指令模板】**：

```
请为以下内容设计比喻：

【本体】
[要描写的内容]

【要求】
- 避免常见喻体
- 喻体要有新的联系
- 符合当前情绪基调

【头脑风暴】
请列出3-5个可能的喻体，然后选择最佳：
1. [喻体1]
2. [喻体2]
3. [喻体3]
...

【最终选择】
喻体：[选择]
理由：[为什么这个最好]
```

### 4.2 何时不用修辞

**【原则】**：
1. 情绪高潮时不用——行动和对话更有力
2. 信息密集时不用——修辞会分散注意力
3. 重复表达时不用——简洁更有效

**【判断方法】**：
- 这句话的核心是"说什么"还是"怎么说"？
- 如果是"说什么"，少用修辞
- 如果是"怎么说"，可以用修辞

## 五、叙述视角控制

### 5.1 视角选择

**【第一人称】**
- 优势：深度代入、主观真实感
- 局限：只能知道"我"知道的
- 适用：内心戏丰富、需要代入感

**【第三人称限知】**
- 优势：有一定代入感+客观叙述
- 局限：只能展示视角人物知道的信息
- 适用：大多数小说

**【全知视角】**
- 优势：可以展示所有人的内心
- 局限：容易"说教"、代入感弱
- 适用：复杂群像、史诗叙事

### 5.2 避免越界

**【常见越界类型】**：

| 越界类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 第三称写内心 | "凶手其实是他"（读者不该知道） | 只写角色的观察 |
| 第一人称写不在场 | "他走进房间"（但"我"不在场） | 改变叙事方式 |
| 突然全知 | 第三人称写了所有人的想法 | 保持限制 |

**【AI指令模板】**：

```
请检查以下段落的视角问题：

【原文】
[粘贴]

【当前视角】
[第一人称/第三人称限知/全知]

【视角规则】
- [列出当前视角的限制]

【越界检查】
- [ ] 是否写了视角人物不知道的信息？
- [ ] 是否写了不在场的内容？
- [ ] 是否有突然切换视角？

【修正建议】
如有越界，请给出修正方案。
```

### 5.3 视角人物死亡的写法

**【问题】**：视角人物死亡后如何继续叙事？

**【方法】**：
1. 切换到其他存活角色的视角
2. 使用"后来"的叙事方式（"他不知道的是……"）
3. 使用倒叙/插叙

**【禁止】**：
- 视角人物死亡后仍以其视角叙述

## 六、对话写作法

### 6.1 对话推动法

**【目标】**：让每个对话都有存在的意义

**【对话功能检查】**：

```
对话必须满足至少一个：
□ 推动剧情（新信息、新事件）
□ 揭示性格（通过说话方式）
□ 改变关系（关系在此刻变化）
□ 制造冲突（矛盾产生或升级）

禁止的对话：
□ 无意义的寒暄
□ 信息重复
□ 可以用动作代替
```

### 6.2 对话-叙述平衡

**【问题】**：对话太多像剧本，对话太少像自传

**【平衡建议】**：
- 关键对话完整写
- 非关键对话可用"他说"、"他们聊了很久"一笔带过
- 用动作和表情丰富对话

**【混合写法】**：
```
"你要走了？"她没有抬头，手里的针线活儿也没停。
"嗯。"他站在门口，犹豫了一下，"……明天见。"
```

### 6.3 对话标签设计

**【标签使用原则】**：
- 少用"他说""她说"
- 用动作替代标签
- 用内容区分说话者

**【多样化标签】**：
- 动作型："他放下茶杯"
- 情绪型："他的声音沉下来"
- 内容型：读者能通过内容区分时不加标签

## 七、信息密度控制

### 7.1 每句话承载信息

**【目标】**：删除空转句，每句话都有价值

**【空转句识别】**：

| 空转类型 | 示例 | 问题 |
|---------|------|------|
| 重复信息 | "他很紧张，他的心跳加速" | 信息重复 |
| 废话填充 | "然后……然后……" | 无信息 |
| 旁观描写 | "他走进房间，房间里有一张桌子" | 无意义 |
| 情绪直说 | "她感到悲伤" | 可用动作替代 |

### 7.2 信息密度检查

**【方法】**：每段话问自己
- 这句话告诉读者什么？
- 如果删除这句话，会影响理解吗？
- 这句话可以用更少的字表达吗？

**【AI指令模板】**：

```

[粘贴]

【逐句分析】

- 第2句：信息点：[...] 是否必要：[是/否]
...

【删除建议】
```

## 八、展示vs告知

### 8.1 展示法核心

**【核心原则】**：
- 不要告诉读者"是什么"
- 展示让读者自己得出结论

**【对比示例】**：

| 告知（错误） | 展示（正确） |
|-------------|-------------|
| "他很紧张" | "他的手在发抖，烟灰落了一地" |
| "她很生气" | "她的声音升高了八度，玻璃杯被摔在地上" |
| "他是好人" | "他把仅有的面包让给了孩子" |
| "房间里很冷" | "窗户上结着霜，她呼出的气清晰可见" |

### 8.2 何时用告知

**【可以用告知的情况】**：
1. 不重要的次要信息
2. 过渡性叙述
3. 作者刻意强调时（但要克制）
4. 时间跨度大的叙述

**【禁止用告知的情况】**：
1. 角色性格——必须展示
2. 角色情感——必须展示
3. 重要主题——最好展示

### 8.3 展示法训练

**【转化练习】**：

```
告知句：[改写]
↓ ↓ ↓
展示句：[结果]

练习：
"他很害怕" → "他往后退了一步，背抵着墙"
"她很开心" → "她笑起来，眼睛弯成月牙"
"这是个危险的地方" → "警示牌上的字已经褪色，但骷髅头标志还清晰可见"
```

**【AI指令模板】**：

```
请将以下告知句改为展示句：

【告知句】
1. [句子]
2. [句子]
3. [句子]

【要求】
- 用动作/对话/环境替代直接描述
- 保持原有信息
- 增加感官细节

【输出】
展示句1：[写出]
展示句2：[写出]
展示句3：[写出]
```

## 九、常见问题与修复

### 问题1：用词重复

**表现**：同一个词出现超过3次

**修复**：
1. 制作同义词表
2. 用代词替代
3. 用动作替代

### 问题2：句式单调

**表现**：全篇句子长度相同

**修复**：
1. 长句后加短句
2. 短句后加长句
3. 根据情绪调整

### 问题3：修辞堆砌

**表现**：到处都用比喻、拟人

**修复**：
1. 删除可有可无的修辞
2. 保留最精彩的比喻
3. 用具体细节替代修辞

### 问题4：视角混乱

**表现**：写了视角人物不知道的内容

**修复**：
1. 确定当前视角
2. 列出视角限制
3. 检查每一句话

### 问题5：对话太满

**表现**：通篇都是"说"

**修复**：
1. 用动作替代标签
2. 用"省略"替代过渡对话
3. 用叙述替代不重要的对话

### 问题6：信息密度低

**表现**：废话连篇

**修复**：
1. 删除重复信息
2. 合并相近信息
3. 删除旁观描写

### 问题7：全是告知

**表现**：直接告诉读者结论

**修复**：
1. 列出所有"告知句"
2. 转化为展示
3. 用动作/对话替代
