## 三、关键发现

1. **fill-chapter新增了stream版本** — 前端StepFill的AI填充应优先改为SSE流式，体验提升巨大
2. **续写/去AI味是全新能力** — StepFill的AI悬浮菜单需要对接这2个API
3. **outline-tree是多级结构** — StepOutline目前是单层，需改树形展开
4. **characters有完整CRUD+关系图谱** — 需要新页面，但共创流程中也可以通过batch-query加载角色上下文
5. **fill-event支持use_dna** — 写作DNA功能可用在填充时
6. **190个端点 vs 前端已对接16个** — 覆盖率约8%，大量功能待对接

## 四、建议对接节奏

1. **第一轮（1-2天）**：P0 — 流式输出 + 续写 + 去AI味，StepFill体验质变
2. **第二轮（2-3天）**：P1+P2 — 素材设定库 + 大纲版本，完善创作工具链
3. **第三轮（1-2天）**：P3+P4+P5 — DNA + 纠错 + 版本管理
4. **第四轮**：P6-P8 — 知识库/创作引擎/积分，按需推进
