### 1.1 顾深（男主角）

| 项目 | 内容 |
|------|------|
| **姓名** | 顾深 |
| **性别** | 男 |
| **年龄** | 36岁（2040年） |
| **国籍** | 中国 |
| **职业** | 天文物理学家、量子通讯专家 |
| **身份** | "黑洞图书馆"计划核心成员、鲲鹏号首席科学家 |
| **家庭** | 丧妻，有一子（6岁），与母亲同住 |

#### 外貌特征

- **身高**：178cm
- **体型**：偏瘦，长期伏案工作导致略有驼背
- **眼睛**：深邃、温和，眼底常有淡淡青黑（失眠的痕迹）
- **眉毛**：浓密但不太修整，随意自然
- **发型**：短发，有些凌乱，鬓角略长，耳边常有几根白发（35岁后出现）
- **标志特征**：左手无名指上有一道淡淡的戒指痕（妻子离世后仍保留的习惯）
- **穿着**：日常穿休闲衬衫+牛仔裤，工作中穿白大褂或军装便服

#### 性格特质

| 维度 | 描述 |
|------|------|
| **核心性格** | 内敛、理性、责任感强、情感克制 |
| **正面特质** | 专注、坚定、温柔、有担当 |
| **负面特质** | 不善表达、习惯性逃避、过度自责 |
| **矛盾点** | 外表冷静但内心汹涌，理智到近乎冷漠，但为爱愿意赌上一切 |
| **性格成因** | 科研工作塑造了理性思维；妻子离世让他学会克制；父亲缺席让他渴望守护家庭 |
| **成长方向** | 从"独自承担"到"学会信任团队"，从"逃避遗憾"到"直面选择" |

#### 背景故事

**出生与成长**：
- 1984年出生于江苏南京
- 父亲是军官，常年不在家；母亲是中学教师
- 从小由母亲独自抚养，与父亲关系淡漠，但骨子里继承了家国情怀
- 高中时就对物理学产生浓厚兴趣，尤其痴迷量子力学

**教育与事业**：
- 18岁保送北京大学物理系
- 25岁获得麻省理工学院天体物理博士学位
- 28岁回国，加入国家航天局量子通讯项目组
- 32岁成为"黑洞图书馆"计划首席科学家

**婚姻与家庭**：
- 30岁认识妻子林晚舟（图书馆偶遇）
- 32岁结婚，33岁儿子出生
- 34岁妻子确诊（长期积劳+交通事故旧伤复发），35岁妻子离世
- 此后独自抚养儿子，与母亲互相照应

**入伍原因**：
- 2039年接到保密入伍通知，成为特殊技术人员
- 表面是"为国家做贡献"，实际是想"找到某种方法再见妻子一面"
- 他知道"黑洞图书馆"计划可能触及"时空"的边界

#### 核心动机

**表层动机**：完成国家任务，为人类探索宇宙
**深层动机**：寻找"再见妻子一面"的可能性
**隐藏动机**：保护儿子，给儿子一个"完整的家"

#### 人物弧光

| 阶段 | 状态 | 变化 |
|------|------|------|
| **开篇** | 克制、压抑，用工作麻痹自己 | — |
| **训练期** | 被队友温暖，重新感受团队 | "我不是一个人" |
| **任务中** | 情感爆发，试图改变命运 | "我想救她" |
| **回归后** | 发现历史改变，身份迷失 | "我到底是谁" |
| **结局** | 接受选择，承担后果 | "无论哪个世界，我都会找到你" |

#### 关键台词

> "我不是英雄。我只是一个想见妻子的普通人。"

> "儿子问我妈妈是什么样的人。我说她像天上的月亮——你知道吗，在黑洞附近，你看见的月亮，可能是一秒钟前的她。"

> "如果能重来一次，我会在她累的时候，逼她休息。"

> "我改变了历史……代价是她从未存在。"

> "这次，我不会错过你。"

#### 标志性动作

- **紧张时**：揉捏左手无名指的戒指痕
- **思考时**：推眼镜（实际上没戴眼镜的习惯性动作）
- **说谎时**：视线向左下方飘移
- **想念妻子时**：看着窗外的月亮发呆

### 1.2 林晚舟（女主/妻子）

> **角色说明**：林晚舟在故事中以"记忆"和"平行世界"两种形态出现。正文中她是"过去"和"平行世界"中的活人；开篇和结局中，她是男主记忆中的人物。

| 项目 | 内容 |
|------|------|
| **姓名** | 林晚舟 |
| **性别** | 女 |
| **年龄** | 32岁（离世时）/ 记忆中的29岁 |
| **国籍** | 中国 |
| **职业** | 儿童文学作家 |
| **家庭** | 丈夫顾深、儿子顾念（6岁） |

#### 外貌特征

- **身高**：165cm
- **体型**：纤细，瘦弱（长期劳累导致）
- **眼睛**：明亮、爱笑，眼角有细细的笑纹
- **眉毛**：淡淡的弯月眉，不太化妆
- **发型**：长发过肩，自然黑色，偶尔扎马尾
- **标志特征**：左耳垂有一颗小痣；笑起来有两个浅浅的酒窝
- **穿着**：舒适为主，棉麻材质，喜欢米色、浅蓝等柔和颜色

#### 性格特质

| 维度 | 描述 |
|------|------|
| **核心性格** | 温柔、坚强、乐观、隐忍 |
| **正面特质** | 善良、体贴、包容、幽默 |
| **负面特质** | 太过坚强、不愿麻烦别人、透支自己 |
| **矛盾点** | 表面开朗但内心敏感；把所有温暖给了别人，唯独忘了自己 |
| **性格成因** | 童年家庭条件普通，养成了独立的性格；热爱儿童文学让她充满童心 |
| **成长方向** | 从"独自承担"到"学会依靠"（但未能完成） |

#### 背景故事

**出生与成长**：
- 1991年出生于浙江杭州
- 父母是普通工薪阶层，家教开明
- 从小喜欢讲故事，是孩子堆里的"故事大王"
- 大学就读中文系，梦想成为儿童文学作家

**事业**：
- 毕业后在出版社工作，负责童书编辑
- 业余时间创作绘本，笔名"小舟"
- 生前最后一本《月亮的孩子》出版于她离世后一个月

**婚姻**：
- 与顾深相识于图书馆（2030年）
- 婚后事业进入高峰期，出版了6本绘本
- 生子后一边带孩子一边工作，身体逐渐透支
- 从不向丈夫抱怨累，总说"没事，我能行"

**离世**：
- 有过交通事故（左腿旧伤）
- 生子后长期睡眠不足
- 积劳成疾+旧伤复发，离世前一周还在写书稿
- 顾深在她病床前承诺"我会照顾好儿子"

#### 核心动机

**表层动机**：完成手头的绘本创作
**深层动机**：给儿子留下"妈妈的故事"，让他长大后知道妈妈有多爱他

#### 经典台词

> "老公，你说月亮上真的有兔子吗？"

> "念念问起来，你就说我去很远的地方出差了。别让他知道……"

> "别哭。我只是累了，想休息一会儿。"

> "照顾好自己和儿子，别只顾着工作……记得按时吃饭。"

> "如果有来生，我还要嫁给你。"

#### 平行世界中的林晚舟

在"新世界"（平行世界）中：
- 林晚舟从未与顾深相识
- 她仍然活着，在杭州一家出版社工作
- 仍然是那个温柔、坚强、爱讲故事的女孩子
- 顾深需要在图书馆"重新认识"她
