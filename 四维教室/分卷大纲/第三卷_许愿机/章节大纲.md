## 二、每章详细大纲

### 第1章：水坑

**字数**：约3200字
**前景/内层比例**：20% / 80%
**节奏定位**：起——强钩子，"踩水坑，不够"

**前景线（约600字）**

老师开场，介绍今天要讲的故事。

```
"今天，我们来看一个关于'想要'的故事。"

"想要？"球球问。

"是的。这个故事里，有一个人想要一样东西。
她没有得到。但她最后明白了另一件事。"

"什么东西？"泡泡问。

"想要。"老师说，"你们想要过什么吗？"

球球："我想吃糖！"

卡卡："我想变成超人。"

暖暖："我想……我想让奶奶的腿不疼。"

石头："我想知道答案。"

泡泡："我想……我想许愿！"

老师笑了一下："很好。"

老师看向芽芽："你呢？"

芽芽没说话。
```

**内层线（约2600字）**

**开头爆款（500字）**

```
这个世界没有雨。

天空永远是灰的，地上永远是干的。没人觉得不对，
因为没人见过别的样子。

阿芽见过——在画上。巷口那幅画，小孩踩进水坑，
水花飞到头顶。

她想踩。

今天，贩卖机旁边有一滩水。不知道哪来的，就比一只鞋大一点。

阿芽踩了一脚。

水花溅到鞋面上。

不够。

使劲踩，还是那么点。

她抬头看贩卖机。屏幕亮着，三个空格子一闪一闪。

旁边贴着纸：投一枚，许一次。
```

**阿芽的犹豫（1000字）**

阿芽看着那三个空格子。

她知道那是什么意思。那三个格子是她身体里的东西——每个人生下来都有三个。

她九岁了。一个都没用过。

奶奶说过："格子不是用来要的，是用来用的。"

阿芽不懂这句话。但她记住了。

她看了看水坑。不大。但她想要更大。

**最后的选择（800字）**

阿芽把手放在投币口。

她的手指冰凉。机器也是冰凉的。

"我要——"

她停了一下。

"我要踩大水坑。"

机器震动了一下。屏幕上的三个格子，有一个亮了起来。

然后灭了。

什么都没发生。

至少——暂时什么都没发生。

**章节结尾钩子**

天空还是灰的。水坑还是那么小。阿芽站在那里，不知道该不该把手收回来。

然后，第一滴雨落在了她的鼻尖上。

### 第2章：灰色的世界

**字数**：约3200字
**前景/内层比例**：15% / 85%
**节奏定位**：承——建立灰色世界日常

**前景线（约450字）**

幼崽讨论灰色世界。

```
球球："为什么世界是灰色的？"

老师："因为可能性变少了。"

卡卡："什么是可能性？"

老师："如果我说，明天可能下雨，也可能不下雨。
这就是两种可能性。"

泡泡："哦！所以以前有下雨的可能性！"

老师："是的。现在没了。"
```

**内层线（约2750字）**

**灰色世界的日常（800字）**

灰色世界没有雨。

阿芽从小就没见过雨。她只在书上看到过这个词，但不知道那是什么感觉。

奶奶说："雨是凉的。会打湿头发。会让人想躲起来。"

阿芽问："那为什么有人想要雨？"

奶奶说："因为躲起来的时候，会觉得世界是安全的。"

阿芽不太懂。

**画里的小孩（1000字）**

阿芽经常看那幅画。

画里的小孩站在水坑里，水花溅到头顶。背景不是灰色的——背景是蓝色的。

"那是什么？"阿芽问过奶奶。

"那是以前。"奶奶说，"以前的世界不是灰色的。"

"为什么变成灰色的了？"

奶奶没有回答。

阿芽就自己看那幅画。她看着那个小孩的动作，看着水花的高度。她想：那个小孩在做什么？

踩水坑。

她想踩。

**灰雾（950字）**

在灰色世界里，有一种东西叫"灰雾"。

灰雾不是普通的雾。它是许愿的痕迹——每一次许愿，都会在空气中留下一点灰色。

大多数人看不见灰雾。但阿芽有时候能看见。

今天，贩卖机旁边有一点灰蒙蒙的东西。很小。不仔细看根本看不见。

阿芽不知道那是什么。

她只知道，她刚才许了一个愿。

### 第3章：格子还是满的

**字数**：约3200字
**前景/内层比例**：15% / 85%
**节奏定位**：承——建立阿芽人设

**前景线（约450字）**

幼崽讨论"格子"是什么。

```
泡泡："三个格子？她想许什么愿望？"

老师："她还没想好。"

卡卡："那她为什么要许愿？"

老师："因为她看到了水坑。"

暖暖："就因为水坑？"

老师停顿了一下："有时候，'想要'不需要理由。"
```

**内层线（约2750字）**

**阿芽的格子（800字）**

阿芽的格子还是满的。

三个。都在。

奶奶的格子不是。奶奶用掉了两个：第一个给了阿芽出生，第二个给了阿芽爸爸的病。

"格子不是用来要的，是用来用的。"奶奶常说。

阿芽问："什么是'用'？"

奶奶说："用就是——你知道自己要什么，然后用它去换。"

阿芽问："那如果不知道要什么呢？"

奶奶说："那就等。等到了就知道了。"

阿芽等了很久。她不知道自己要什么。

直到她看见那滩水。

**阿芽和奶奶（1000字）**

阿芽回到家，奶奶在做晚饭。

"今天怎么回来晚了？"奶奶问。

"……没什么。"阿芽说。

奶奶看了她一眼，没有追问。

阿芽坐在桌边，看着奶奶忙碌的背影。奶奶老了，背有点驼。奶奶的腿不好，走路慢。

奶奶曾经用过一个格子想让腿好起来。但没有。

"格子不是万能的。"奶奶说，"格子只能换一样东西。换完就没有了。"

阿芽问："那您后悔吗？"

奶奶说："不后悔。我换了你爸爸。"

阿芽低下头。

**今天发生的事（950字）**

阿芽躺在床上，睡不着。

她一直在想那滩水。

她踩了一脚。不够。她又踩了一脚。还是不够。

然后她许了一个愿。

"我要踩大水坑。"

然后什么都没发生。至少她以为什么都没发生。

然后——

第一滴雨落在了她的鼻尖上。

阿芽坐起来。

她走到窗边，看向窗外。

天空还是灰色的。但有什么不一样了。

空气里有一种……湿的东西。

她深吸一口气。

下雨了？

### 第4章：就这样

**字数**：约3200字
**前景/内层比例**：15% / 85%
**节奏定位**：转——第一次选择

**前景线（约450字）**

幼崽讨论"直接许愿"。

```
卡卡："她可以直接许愿让水坑变大啊！"

老师："她确实许了。"

泡泡："那她为什么还要踩？"

老师："因为她不知道许愿会变成什么样。"

暖暖："……变成什么样？"

（教室里安静了）
```

**内层线（约2750字）**

**犹豫了很久（1000字）**

阿芽站在贩卖机旁边。

不是今天。是之前。在许愿之前。

她站在那里很久。

很久。

奶奶说过："格子不是用来要的，是用来用的。"

但什么是"用"？奶奶每次说这句话的时候，眼神都很复杂。

阿芽不懂。但她记住了那句话。

她看着自己的三个格子。它们在胸口的位置。一跳一跳的。

她看着贩卖机。屏幕亮着，三个空格子一闪一闪。

旁边写着：投一枚，许一次。

**她在想什么（800字）**

阿芽在想很多事。

她想：如果我许愿，我会不会后悔？

她想：如果我不许愿，我会不会后悔？

她想：如果我许愿变大水坑，奶奶会怎么说？

她想：如果我许愿让奶奶的腿好起来，我会怎么说？

她想了很多。

但她不知道自己要什么。

直到——

她看见那滩水。

很小的水。比一只鞋大一点。

她踩了一脚。

水花溅到鞋面上。

不够。

她想：够了就够了。

不需要更大。

**就这一次（950字）**

阿芽把手放在投币口。

机器冰凉的。

"我要——"

她停了一下。

然后她说了出来：

"我要踩大水坑。"

机器震动。屏幕上的格子亮了一个，然后灭了。

阿芽等了几秒。

什么都没发生。

水坑还是那么小。天空还是灰的。

她正准备走——

然后第一滴雨落在了她的鼻尖上。

### 第5章：第一个愿望

**字数**：约3200字
**前景/内层比例**：15% / 85%
**节奏定位**：高潮起——第一次反转

**前景线（约450字）**

泡泡笑，然后沉默。

```
泡泡："就这？她许的愿望就是这个？踩水坑？"

老师点头。

泡泡："她可以许愿变成公主！或者变成超人！"

老师："……"

（雨的声音从故事里传出来）

泡泡："……"

（泡泡沉默了）
```

**内层线（约2750字）**

**雨来了（800字）**

第一滴。第二滴。第三滴。

然后——哗——

雨来了。

不是小雨，是大雨。是很大的雨。水从天上倒下来，打在阿芽的脸上，打在她的头发上，打在她的衣服上。

阿芽愣在原地。

她从来没有被雨淋过。

凉凉的。湿湿的。打在脸上有点疼，但也很舒服。

她抬头看天。灰色的天空，现在变成了——不知道什么颜色。水从天上落下来，模糊了一切。

**笑疯了（1000字）**

阿芽笑起来了。

她从来没有这么笑过。

她站在雨里，水花溅起来，溅到她的头顶，溅到她的脸上。她张开手臂，转圈圈。

"哈！哈哈哈！"

水坑变大了——不对，不是水坑变大了。是地上全是水。不管她踩哪里，都是水坑。

她使劲踩。用力踩。

水花溅起来，溅到她的头顶，溅到她的肩膀，溅得到处都是。

就像那幅画里的小孩一样！

不，比那幅画里的小孩还要——

然后她笑不出来了。

**不对（950字）**

雨太大了。

太大了。

阿芽看不清前面。水从天上砸下来，砸得她睁不开眼睛。

她伸手想擦脸，但水马上又糊住了她的眼睛。

脚下——脚下踩到什么了？地上有东西在动。

不是水坑。

是——

阿芽低头看。

水里有什么东西浮出来。

不是鱼。不是石头。

是——颜色。

各种颜色。碎片化的、没有形状的颜色。像照片烧掉时的那一瞬间。

然后雾来了。

### 第6章：笑疯了

**字数**：约3500字
**前景/内层比例**：15% / 85%
**节奏定位**：高潮——情绪引爆，笑和恐惧同秒

**前景线（约500字）**

暖暖问，泡泡问。

```
暖暖："她……害怕吗？"

老师："你觉得呢？"

暖暖："她刚才还在笑……"

泡泡："那她后来为什么害怕了？"

老师停顿。

泡泡："老师？"

老师："她看到了一些东西。"
```

**内层线（约3000字）**

**雾里的东西（1000字）**

雾很浓。

阿芽什么都看不见。

她站在原地，不敢动。水还在下，但雾比水更让人害怕——因为雾里有什么东西。

她听到了声音。

不是雨的声音。是别的东西的声音。像很多人在说话，但又说不出具体的词。像很多画面在闪，但没有具体的形状。

这是"可能性残影"。

被撕碎的可能性不会完全消失。它们会在雾中留下残影。

阿芽不知道自己看到了什么。她只知道——那很可怕。

**笑和恐惧同秒（1000字）**

阿芽的脚还在水里。

水很舒服。冰凉的，柔软的，踩下去会有水花。

但她的心很慌。

她想继续踩。她想笑。她想大叫。

但她也在害怕。

两件事同时存在。同时发生。

这就是"灰色世界的代价"——你想要一个东西，你要付出另一个东西作为代价。

你笑的同时，你也要承担害怕。

**水坑变大了（1000字）**

不知道过了多久。

雾散了。

雨还在下，但小了一点。阿芽能看清周围了。

地上全是水。不只是水坑——是河流。是池塘。是——水世界。

她的愿望实现了。

她有了一个"大水坑"。

比她想象的还要大。

大到她现在站在水里，水已经到了她的膝盖。

她低头看自己的脚。水很浑浊，看不清下面有什么。

然后她听到了一个声音。

从很远的地方传来。像是有人在喊。

"阿芽——"

是奶奶的声音。

"阿芽，你在哪里——"

阿芽张了张嘴，想回答。

但她发现自己说不出话。

### 第7章：雾来了

**字数**：约3500字
**前景/内层比例**：20% / 80%
**节奏定位**：高潮承——强化细节，三种影响施压

**前景线（约650字）**

石头分析。

```
石头："她不是故意的。"

暖暖："但结果已经这样了……"
```
