## 二、每章详细大纲

### 第1章：墨城的星星

**字数**：约3500字
**前景/内层比例**：30% / 70%
**节奏定位**：起——建立世界与主角

**前景线（老师+幼崽，1000字）**

老师开场，介绍今天要讲的故事。

```
"今天，我们来看一个关于画家的故事。"

"画家？"球球问。

"是的。这个画家很特别。她不是画在纸上的画家，
她是画在空气中的画家。她画出来的东西，会变成真的。"

"哇！"暖暖说。

"她叫小弦。"老师说，"十二岁。"

（幼崽们的反应：十二岁？比我们大一点！）
```

**内层线（墨城与小弦，2500字）**

**墨城开场（800字）**

介绍墨城：人们可以用手画画，画出来的东西会变成真的。
- 画一颗星星 → 天上多了一颗星星
- 画一座桥 → 真的可以走过去
- 墨是画画的"颜料"，从地底开采

**小弦出场（1000字）**

六岁画出第一颗星星。八岁画出一条河。十岁画出飞行器。

小弦站在城墙上，看着自己画的星星挂在天上。那颗星星是墨城夜空中最亮的一颗。

她很骄傲。

**危机背景（700字）**

但最近，墨城的气氛变了。

供应处的墨越来越少。普通人开始担心。

小弦听说这件事，但觉得和自己关系不大。她有足够多的墨。

**本章幼崽打断设计**

```
球球：她几岁？

老师：十二岁。

球球：比我们大不了多少！

卡卡：但是她画得那么厉害……

老师：是的。很厉害。
```

**暗线埋入**：无
**章节结尾钩子**

小弦放学回家，路过供应处。她看了一眼门口排队的普通人，没有多想。

### 第2章：最聪明的孩子

**字数**：约3500字
**前景/内层比例**：20% / 80%
**节奏定位**：承——建立主角魅力与孤独

**前景线（600字）**

幼崽讨论小弦为什么这么厉害。

```
球球：她为什么不累？画那么多东西。

老师：你们觉得呢？

暖暖：她喜欢吧？

老师：喜欢，也是一种力量。
```

**内层线（2900字）**

**小弦的日常（800字）**

小弦在画室学习，但她早就超越了画室教的内容。

老师让她画一个苹果，她画了一个完美的苹果——然后把它变成了真的苹果，分给同学吃。

同学：谢谢小弦！

小弦：嗯。（淡淡的）

**天才的孤独（1000字）**

小弦发现，她和其他孩子不太一样。

她画的画，总比别人好。

她学的知识，总比别人快。

她觉得，有些东西很简单，别人却怎么都学不会。

她不理解：为什么你们不懂呢？明明很简单啊。

但她也发现，太厉害了有时候很孤独。

没人能真正理解她的画。

**朋友：阿墨（500字）**

小弦有一个朋友，叫阿墨。阿墨是三级画师，画力中等。

阿墨说："小弦，你是我见过最厉害的人。"

小弦说："还好吧。"

阿墨说："但你好像不太开心。"

小弦想了想："没有啊。"

她不知道自己算不算开心。她只知道，她还要画更多的东西。

**长老的邀请（600字）**

长老会议邀请小弦参加。

长老们讨论墨快用完的问题。他们很担心。

小弦听着，忽然开口："我有一个办法。"

所有人看向她。

"画一个黑洞。"小弦说。

**本章幼崽打断设计**

```
暖暖：她好勇敢……万一失败了呢？

老师：……

（老师没有回答）
```

**暗线埋入**：无
**章节结尾钩子**

小弦说出"画一个黑洞"的提议。长老们面面相觑。

"黑洞？"长老甲问，"那是什么？"

小弦说："一种可以产生墨的东西。"

---

### 第3章：墨快用完了

**字数**：约3500字
**前景/内层比例**：20% / 80%
**节奏定位**：转——危机出现，方案提出

**前景线（700字）**

幼崽问"墨快用完了是什么意思"。

```
暖暖：墨快用完了……大家会死吗？

老师：不会死。但会很麻烦。

卡卡：为什么？

老师：因为他们画画需要墨。没有墨，就画不出东西了。
```

**内层线（2800字）**

**墨的危机（800字）**

供应处张贴了新的分配规则：
- 普通人的墨配额减少三分之一
- 画匠的墨配额减少四分之一
- 大画师的墨配额暂时不变

街上开始有人议论。

"墨越来越贵了。"
"深层矿脉的墨质量不好。"
"听说灰越来越多。"

小弦听到这些，但没有在意。她有足够多的墨。

**小弦的观察（1000字）**

但她开始思考一个问题：

如果墨是从地底开采的，那地底的墨会开采完吗？

答案是：会。

小弦在图书馆里找了资料。她发现，墨城的矿脉正在枯竭。深层矿脉是最后的储备。

深层矿脉的墨灰含量高，画出来的东西不稳定。

**小弦的方案（1000字）**

小弦有了一个想法。

如果墨会枯竭，那——墨从哪里来？

宇宙中，最强大的能量来源是什么？

黑洞。

黑洞可以吞噬一切物质，也可以产生新的物质。

如果她能画出一个黑洞，黑洞就能在墨城里产生新的墨。

"我画一个黑洞。"小弦对长老们说，"黑洞会自己生成墨。我们永远不会缺墨。"

长老们沉默。

"你确定吗？"长老乙问，"从来没有人画过黑洞。"

"我知道。"小弦说，"但我能做到。"

**本章幼崽打断设计**

```
暖暖：她好勇敢……万一失败了呢？

老师：……

（老师没有回答）
```

**暗线埋入**：提到"深层矿脉的墨灰含量高"——这是整个故事的起因伏笔
**章节结尾钩子**

长老们决定让小弦试一试。

小弦开始设计她的黑洞。

### ★第4章：墨城的孩子

**字数**：约4000字
**前景/内层比例**：25% / 75%
**节奏定位**：承——普通孩子的视角，小弦路过
**新增章节**

**前景线（900字）**

幼崽讨论小弦和其他孩子的区别。

```
卡卡：她认识那些排队的人吗？

老师：你们觉得呢？

球球：她那么厉害，肯定认识吧？

老师：……继续听故事。
```

**内层线（3100字）**

**场景：墨城街道，傍晚**

小弦放学回家，路过供应处。

门口排着长长的队伍。都是普通人——老人、孩子、妇女。他们拿着空的墨瓶，等着领配额。

小弦看了一眼，继续走。

**小明的视角（1000字）**

小明今年十岁，是三级画师学徒。他排在队伍里，手里拿着一瓶快见底的墨。

小明画不出复杂的东西。他只能画简单的日用品——水杯、勺子、拖鞋。

他很努力地练习，但进步很慢。

"小明，你的墨。"供应处的人递过来一瓶墨。

小明接过墨，看了看："这墨……颜色有点怪。"

供应处的人说："深层矿脉的就这样，凑合用吧。"

小明点点头。他不懂什么是深层矿脉。他只知道，有墨就不错了。

**小弦路过（600字）**

小弦从队伍旁边走过。

小明认出了她。

"看，是小弦！"小明小声对旁边的人说。

"哪个？"

"那个天才小孩！六岁画星星的那个！"

小弦没有看他们。她走过去了，背影很直。

小明看着她的背影，有点羡慕。

他也想变得那么厉害。但他知道，他做不到。

**小弦的想法（500字）**

小弦走远了。

她知道有人在看她，但她没有回头。

她想：他们为什么那么着急？不就是墨吗，等我画好黑洞，就永远不缺墨了。

她觉得很简单的事情，他们为什么要排队？

她不太理解。

但她也没有多想。

因为她还有更重要的事要做——她要设计黑洞。

**小明的回家（500字）**

小明回到家，把墨倒进自己的小瓶子里。

他拿出纸，开始画。

他想画一个小灯笼。上次小弦画的灯笼很好看，他也想试试。

他画了三遍，都失败了。

灯笼不是歪的，就是太小，或者太大。

小明叹了口气，把纸揉成一团。

他想起小弦走路的样子。那么自信，那么骄傲。

他低头看了看自己的画。

"……算了。"他想，"我还是画勺子吧。"

**本章幼崽打断设计**

```
卡卡：她不认识他们吗？

老师：……她知道他们在等。但她有自己的事要做。

泡泡：什么事？

老师：画黑洞。
```

**暗线埋入**：无
**章节结尾钩子**

小弦回到画室，开始画黑洞的设计图。

小明躺在床上，想着明天还要排队领墨。

### ★第5章：第一次试画

**字数**：约4000字
**前景/内层比例**：20% / 80%
**节奏定位**：高潮起——小弦的试画过程
**新增章节**

**前景线（800字）**

幼崽好奇小弦是不是一下就成功了。

```
石头：她第一次画黑洞就成功了吗？

老师：不是。她先试了一次。

球球：试画是什么？

老师：就是先画一个小的，看看行不行。

卡卡：那成功了吗？

老师：……你们觉得呢？
```

**内层线（3200字）**

**小弦的决定（600字）**

小弦决定先试画。

"不能直接画大的，"她想，"要先确认我的设计是对的。"

她决定画一个微型黑洞——只有拳头那么大。

如果成功了，她再画真正的黑洞。

**试画准备（800字）**

小弦去找画室的老师申请墨。

"微型黑洞试画？"老师看了看她的设计图，"你知道这有多危险吗？"

"我知道。"小弦说，"所以我只画一点点。如果出问题，画室的墙壁会吸收。"

老师想了想，批准了。

但老师给了她一个条件：只能使用库存里的"老墨"——那些来自浅层矿脉、质量最好的墨。

小弦答应了。

**第一批墨（800字）**

供应处给了小弦一批老墨。

这些墨很黑，很纯，没有一点灰色。

小弦打开瓶子，闻了闻。很香。

"这就是最好的墨。"她想。

她把墨装进自己的容器，开始试画。

**试画过程（1000字）**

小弦站在画室中央，闭上眼睛。

她回想黑洞的设计：事件视界、能量注入、引力场维持。

她睁开眼睛，伸出手。

她的手指在空中移动，画出细细的线条。

线条在空中悬浮，越来越复杂——一个完美的球形，线条越来越细，越来越精确。

这是微型的事件视界。

小弦打开墨容器，把一滴墨滴在视界中心。

墨在视界里滚动，开始发光。黑色的光，越来越亮。

小弦的眉头紧锁。她在用全部的注意力控制墨的流动。

然后——

一个拳头大小的黑洞成形了。

它悬浮在空中，完美无缺。

小弦笑了。

成功了！

**试画结果（500字）**

小弦收回手，黑洞消失了。

她检查了一下：只用了十分之一的墨。

"太棒了。"她想，"设计是对的。只要按照这个方法，我就能画出一个真正的黑洞。"

她不知道的是——试画用的是最好的墨。

而真正的黑洞，她要用的是第二批墨。

**本章幼崽打断设计**

```
石头：所以她成功了？

老师：成功了。但……

卡卡：但什么？

老师：用的墨不一样。

球球：哪里不一样？

老师：继续听。
```

**暗线埋入**：试画用的是"老墨"（质量最好的）——这是重要伏笔，小弦后来用的是含灰的墨
**章节结尾钩子**

小弦开始准备真正的黑洞。

她申请了更多的墨配额。

长老们批准了。

### ★第6章：第二批墨

**字数**：约4000字
**前景/内层比例**：20% / 80%
**节奏定位**：高潮承——阿墨发现墨的质量问题
**新增章节**

**前景线（800字）**

幼崽讨论阿墨的角色。

```
泡泡：阿墨是谁？

老师：阿墨是小弦的朋友。

泡泡：她很厉害吗？

老师：她不是最聪明的。但她很细心。

卡卡：细心有什么用？

老师：有时候……细心的人能看见别人看不见的东西。
```

**内层线（3200字）**

**阿墨的工作（700字）**

阿墨在供应处帮忙。

她不是大画师，但她的工作是分类和记录墨的质量。

这是枯燥的工作，但阿墨做得很认真。

她会把不同批次的墨分开标记，记录每一批的灰含量。

今天，她检查了新到的一批墨。

"这批墨……"她皱了皱眉，"灰含量有点高。"

她看了看标签：深层矿脉，第三层。

"这批墨不能直接供应给画师。"她在记录本上写道，"需要单独存放，检测后再分配。"

**阿墨听说小弦的申请（600字）**

阿墨听说小弦申请了额外的墨配额。

"是画黑洞用的吗？"她问同事。

"是的。大画师配额加额外申请，全部批准了。"

阿墨想了想，问："那批深层矿脉的墨呢？"

同事翻了翻记录："那批？已经分发出去了。"

"分发给谁？"

"大部分分给普通人了。但有一批……"同事看了看单子，"调给了小弦。"

阿墨愣住了。

"哪一批？"

"就是那批灰含量高的。"

**阿墨的发现（800字）**

阿墨决定去查一查。

她跑到仓库，找到了发给小弦的那批墨。

她打开一瓶，看了看。

- **内层≥80%**：故事主体占绝对优势
- **幼崽铁律**：不说大人话、用自己回答自己
