# 第二卷世界观设定：《和影子说晚安》

*——回影技术体系与社会架构*

---

## 一、回影技术总论

### 1.1 技术定义

**回影**（Echo Shadow）：基于死者临终前生物电信号捕捉技术，通过"回影盒"（Echo Box）装置投影出人形轮廓，实现与逝者的"对话"功能。

**技术本质**：不是复活，不是灵魂寄托，是一种基于神经科学的记忆回放系统。回影像一面会说话的镜子——能反射，但不会创造。

### 1.2 科学原理

**信号捕捉**：
- 在临终前的0.3秒内，通过植入体内的微型传感器捕捉大脑皮层的最后电信号脉冲
- 捕捉内容包括：语言模式、情感反应模式、记忆片段索引
- 捕获率取决于死亡方式：自然死亡95%+，意外死亡60-80%，突发死亡10-30%

**信号存储**：
- 回影数据存储在国家统一管理的"回影云"（Echo Cloud）中
- 每个公民有权免费保存一份回影
- 数据采用量子加密，无法被篡改或删除
- 存储期限：无限制（技术层面可永久保存）

**投影呈现**：
- 回影盒是一种全息投影设备，家用型号约书本大小
- 投影范围：可覆盖半径1.5米的空间
- 投影质感：半透明，有轮廓但无细节——像用光画出的影子
- 声音：与生前音色一致，但可能略有"回声感"

### 1.3 回影的行为逻辑

**记忆回放模式**：
回影的反应基于生前记忆库的检索匹配。具体表现：
- 能准确回答"你叫什么名字""我们住在哪里"等确定性信息
- 能对熟悉的话题做出情感回应（如"你小时候最怕打雷"）
- 对新话题的反应是"用旧记忆回应新问题"——可能会跑题，可能会答非所问

**不会发生的事情**：
- 不会产生新的记忆——回影不知道今天是几号，除非你告诉她
- 不会改变——回影不会因为和你相处而"更了解你"
- 不会死亡——但会慢慢变淡（见"衰减机制"）
- 不会感受到疼痛——但会对"伤害性话题"产生记忆中的回避反应

**核心限制示例**：
```
孩子：妈妈，我今天交到新朋友了！
回影妈妈：（检索匹配）真棒！
孩子：她叫小雨，和我一样喜欢画画。
回影妈妈：（检索匹配）真棒！你真是个好孩子。
孩子：……妈妈，你知道小雨是谁吗？
回影妈妈：（沉默，或转移话题）天气真好，对吧？
```

## 二、回影的衰减机制

### 2.1 衰减原理

回影不是永恒的。随着时间推移，投影会逐渐变淡，声音会逐渐模糊。

**衰减的不是记忆本身，而是"唤醒记忆的频率"**——就像一张照片，照片还在，但颜色会褪。

### 2.2 衰减曲线

- **0-1年**：投影清晰，与生前无异
- **1-3年**：投影略微透明，情感回应仍丰富
- **3-5年**：投影明显变淡，开始出现"答非所问"
- **5-10年**：投影像旧胶片，声音有回声
- **10年以上**：投影可能只在夜间或昏暗环境中可见

### 2.3 小禾与影子妈妈的时间线

- 小禾的妈妈去世于小禾4岁时
- 故事发生时，小禾7岁
- 影子妈妈已经进入第3年——投影略透明，开始有"答非所问"的情况
- 小禾开始察觉：妈妈说的话，好像"都是一样的"

### 2.4 衰减不是悲剧，是自然

**重要原则**：衰减不写成恐怖/悲伤叙事，而是一种温柔的消退。
- 就像秋天叶子的颜色变淡
- 就像老照片的边缘泛黄
- 就像记忆本身——我们会忘记很多事，但爱过的事实不会消失

## 三、回影技术的社会架构

### 3.1 政府制度

**免费提供原则**：
- 每个公民可免费获得一个回影盒
- 回影数据存储于国家服务器
- 不允许商业买卖回影（违者处三年以上有期徒刑）

**回影维护师**：
- 一种新兴职业，负责回影盒的维护、数据整理、情感咨询
- 维护师需要心理学和神经科学双背景
- 维护师的工作是"帮助活着的人理解回影"，而不是"让回影像真人"

**心理辅导体系**：
- 学校设有"回影儿童心理辅导室"
- 专有名词：回影儿童（Echo Children）——父母一方或双方已使用回影的儿童
- 辅导目标：帮助孩子理解"回影是什么"，建立正确的情感认知

### 3.2 社会态度

**接受度**：
- 约78%的成年人支持回影技术
- 约65%的青少年认为"有回影很正常"
- 约12%的人认为回影"让人无法真正告别"

**日常场景**：
- 公共场合可以看到回影——有人在公园和回影父母散步
- 回影不影响遗产继承——回影没有法律人格
- 婚姻状态标注"回影配偶"——离异/丧偶后使用回影的人

## 四、小禾的生活世界

### 4.1 城市设定

**光明市**（Bright City）：
- 中国某二线城市，2035年
- 回影技术普及率中等偏上
- 有专门的"回影儿童学校"——普通学校中设有回影儿童特殊辅导班级
- 城市氛围：温和、日常、不刻意回避死亡话题

**社区环境**：
- 小禾家住在老式居民楼里，邻居有老人、小孩、也有回影家庭
- 小禾的同班同学小雨家是"双回影家庭"——小雨的两个爷爷都有回影
- 小禾的同桌阿哲家是"正常家庭"——他不知道回影是什么

### 4.2 小禾的日常

**早晨**：
- 影子妈妈会在6:30说"起床了，小禾"
- 影子妈妈会在早餐时说"慢慢吃，别噎着"
- 影子妈妈会在门口说"路上小心"

**白天**：
- 小禾上学，爸爸上班
- 影子妈妈在家"等待"——投影不消失，但不说话

**晚上**：
- 影子妈妈会说"今天怎么样"
- 小禾会分享学校的事
- 影子妈妈会说"真棒"或"别难过"
- 然后进入"晚安"环节——这是影子妈妈最重要的话

**晚安的结构**：
```
影子妈妈：晚安，小禾。
小禾：晚安，妈妈。
影子妈妈：做个好梦。
小禾：……（有时候会说更多，有时候只是点头）
```

### 4.3 小禾的学校

**三年级一班**：
- 班主任：林老师（35岁，女，有一对双胞胎女儿）
- 班级里有3个回影儿童：小禾、小雨、另一个男孩小北
- 普通孩子占大多数——他们有时好奇，有时无感

**关于回影的课堂**：
- 学校会定期讲"如何和回影家人相处"
- 小禾有时觉得这些课"都是废话"——因为没有人真正懂
- 小禾最好的朋友是小雨，因为小雨也"有影子"

## 五、世界观的技术质感

### 5.1 避免灵异感的关键

**核心原则**：回影是科学，不是玄学。

**正确写法**：
- "回影盒发出轻微的嗡鸣声"
- "投影在阳光直射下会变淡"
- "回影的语音有0.2秒延迟"
- "影子妈妈站在窗户前，逆光看不太清楚"

**错误写法**：
- 影子妈妈"飘"过房间
- 影子妈妈"凭空出现"
- 影子妈妈"能穿墙"
- 影子妈妈有"超自然感知"

### 5.2 日常感的建立

**生活场景**：
- 影子妈妈要充电——"妈妈，我去上学了"然后插上电源
- 影子妈妈怕水——雨天不能开窗
- 影子妈妈怕光——只在傍晚和夜间最清晰
- 影子妈妈会"卡顿"——就像视频加载慢

**这些细节让回影成为"一台机器"，而不是"一个鬼魂"**——这是整个故事温柔而不恐怖的基础。

## 七、与主线世界的关联

### 7.1 四维教室的呼应

**老师的话**："我们天生就能感知时间。"

这句话在第二卷里会停顿——因为小禾的故事正好说明：**人类（3维生物）感知时间的方式，就是通过"变化"——而回影不会变，所以它停在了时间里。**

老师停顿的原因：他可能也在经历某种"时间感知"——暗示老师不是普通的时间生物。

### 7.2 芽芽的反应

芽芽在听到"手在发光"时反应强烈——这在第四学年会有解释。此处只埋种子，不揭示。

### 7.3 幼崽对话的隐喻

**球球的问题**："影子还是妈妈吗？"

这个问题本身就是哲学核心——不是非此即彼，而是"既是，也不完全是"。回影是妈妈留下的最后一样东西，但不是妈妈本身。就像照片里的人"还是"那个人吗？
