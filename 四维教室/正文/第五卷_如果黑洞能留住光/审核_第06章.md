## 一、基础表达维度（权重15%）

### 1.1 用词精准度（85分）
**表现**：用词较为精准，动词使用有力量感。
**优点**：
- "她的轮廓勾成金色"——精准且有画面感
- "眼睛底下有什么东西在抖"——情感描写到位
- "系了一个死结，打完之后又打了三个"——细节具体

**扣分点**：
- "脑子里空空的"——略显口语化
- "装不下了、只能让它空着"——比喻略显牵强

### 1.2 句式节奏（88分）
**表现**：句式有变化，长短交错。
**优点**：
- 短句"好。"单独成段，制造停顿
- 长句描写秋千场景，形成情绪积累
- 对话与叙述穿插，节奏感良好

**扣分点**：
- 结尾处节奏略平，可以更紧凑

### 1.3 修辞克制（82分）
**表现**：修辞使用得当但偶有陈旧。
**优点**：
- "像一根细细的火苗"——精准描绘夕阳下的红绳
- "像一面没有波纹的湖"——意象清晰

**扣分点**：
- "眼睛像星星"——大纲中标注的陈旧比喻，本章未使用（好）
- 比喻使用略显谨慎，可以更精妙

### 1.4 叙述视角（90分）
**表现**：视角控制优秀，严格遵循林晓有限视角。
**优点**：
- 始终通过林晓的眼睛观察苏念
- "林晓不知道那是什么。但他知道"——符合孩子认知
- 没有越界写苏念的内心

**扣分点**：无

### 1.5 对话质量（85分）
**表现**：对话自然，有性格区分。
**优点**：
- 苏念的对话简短、克制，符合角色设定
- 林晓的回应犹豫、"好"字单独回应，符合害怕但善良的性格
- 对话有潜台词："我不是来让你帮忙的"——预设了林晓的担忧

**扣分点**：
- 部分对话略显书面语

### 1.6 信息密度（85分）
**表现**：信息密度较高，冗余较少。
**优点**：
- 红绳来历一段承载多重信息（父女关系、分离、联结）
- 反复提及秋千、背影，积累情感

**扣分点**：
- "他知道""他感觉到"等心理提示略显重复

**基础表达维度得分**：85.8分（加权13.4分）

## 三、剧情与结构维度（权重25%）

### 3.1 情感弧线（90分）
**表现**：情感递进清晰。
**优点**：

```
苏念开口 → 林晓后退（害怕本能）→ 苏念解释（红绳）→ 林晓沉默（内心挣扎）→ 林晓答应（选择）
```
